# Portfolio Website

A modern, responsive portfolio website built to showcase my projects, skills, and experience as a developer.


## Highlights

- Built with Next.js and TypeScript using a modular, scalable architecture
- Custom design system with consistent spacing, typography, and layered surfaces
- Fully responsive and accessibility-conscious UI
- Domain-driven structure for reusable and maintainable components
- Optimized for performance and clean developer experience

## Features

- Clean, modern UI with a structured design system
- Fully responsive across mobile, tablet, and desktop
- Fast performance with optimized components
- Clear project showcase and skill sections
- Light and dark mode support
- Reusable and modular component structure

## Tech Stack

- Framework: Next.js
- Language: TypeScript
- Styling: Tailwind CSS
- UI System: Custom design system (tokens, surfaces, shadows)
- Deployment: Vercel

## Project Structure
- /app            Routes and pages
- /components     Reusable UI components
- /modules        Feature-based UI sections
- /styles         Global styles and design tokens
- /public         Static assets

## Design System

The UI is built using a layered design system:

- Surface levels for visual hierarchy
- Responsive typography using clamp
- Consistent spacing and section rhythm
- Shadow system for depth and emphasis

## Getting Started

### Clone the repository

```bash
git clone https://github.com/ivilla15/portfolio.git
cd portfolio
```

### Install dependencies

```bash
pnpm install
```

### Run the development server
```bash
pnpm dev
```
Open http://localhost:3000 in your browser.

### Build for Production
```bash
pnpm build
pnpm start
```

### Deployment
This project is deployed using Vercel.

To deploy updates:
```bash
git push
```

## Customization
You can update:
	•	Projects: /modules/projects
	•	Content: text inside components
	•	Styling: globals.css design tokens
	•	Colors: CSS variables in :root
